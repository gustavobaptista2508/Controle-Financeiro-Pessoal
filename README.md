# GranaOK

**GranaOK** é um sistema web de controle financeiro pessoal desenvolvido em **.NET / Blazor** com estrutura em camadas, utilizando um projeto Web para interface e autenticação, e um projeto Core para regras de negócio, modelos, serviços e acesso ao banco de dados.

O objetivo do projeto é oferecer uma plataforma simples e inteligente para organização de receitas, despesas, contas, categorias, pessoas, lançamentos financeiros, assinaturas e recursos inteligentes com IA.

---

## 📌 Status atual do projeto

O projeto encontra-se em desenvolvimento ativo.

Atualmente, o foco principal está no ambiente **Web**, com integração ao projeto **FinanceiroPessoal.Core** para centralização dos modelos, serviços e persistência de dados.
